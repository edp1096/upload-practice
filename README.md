# 업로드 연습

## 아래 항목들 복붙용

* 드래그&드롭 제어
* 폼업로드, 스트림 업로드
* 임시파일, 이동

* html: `index.html`, `stream.html`
* golang: `uploader`, `uploader-stream` 폴더
* php: `upload.php`, `upload-stream.php`
* 예쁜 style: 그런거 없다

```sh
$ ./caddy.exe file-server --listen :80
```
